Instruction to run python script :
```
usage: scrapping_images.py [-h] [--n N] [--o O] animal

Download images of animals.

positional arguments:
  animal      Name of the animal to download images for.

options:
  -h, --help  show this help message and exit
  --n N       Number of images to download (default: 5).
  --o O       Folder to save downloaded images (default: 'images').
```
